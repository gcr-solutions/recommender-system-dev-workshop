Prepare code and deploy scripts

In this section, you’ll need to fork a github repo from recommender-system-dev-workshop-code repo and clone repo to cloud9 workspace.

  1. Access recommender-system-dev-workshop-code repo, click Fork button

  1. Click user name to fork repo, then you should get a recommender-system-dev-workshop-code repo in your user account. The repo link should be https://github.com/`your github user name`/recommender-system-dev-workshop-code

  1. Clone repo to your cloud9 workspace, please replace your github user name to your github user name.
